# Simple Diagram Dashboard

A React-based dashboard that allows users to upload a circuit diagram image and view a list of electronic components in a sidebar. The project demonstrates React fundamentals, component-based architecture, and responsive UI design.

## Project Objective

This project was developed as a frontend assessment to demonstrate:

- React fundamentals
- Clean component structure
- UI design and layout
- State management using React Hooks
- Basic responsiveness

## Features

### 1. Upload Diagram
- Upload PNG/JPG images
- Drag and drop support
- Image preview
- Display uploaded file name
- Replace uploaded image

### 2. Diagram Viewer
- View uploaded diagram
- Zoom In
- Zoom Out
- Reset view

### 3. Components List Panel
- Sidebar displaying detected electronic components
- Highlight selected component


### 4. Dashboard Layout
- Upload section at the top
- Two-column layout:
  - Left: Diagram Viewer
  - Right: Components List

### 5. Responsive Design
- Works on desktop and tablet
- Layout stacks vertically on smaller screens

## Tech Stack

- React
- JavaScript (ES6+)
- React Hooks
- Vite
- React Dropzone
- CSS
